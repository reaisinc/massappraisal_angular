The Mass Appraisal GIS was created to add soils information to comparable sales and subject properties in order to do regression analysis.

Built using the following software:
  • HTML/Javascript, including jQuery and Angular
  • Node.js
  • PostgreSQL with Postgis
  • R statistical software (installed in PostgreSQL)


Notes

  • You must have a Google account before using this tool.
  • The Mass Appraisal Soils Analyzer was created by Steve Hale.
