The Mass Appraisal GIS provides an easy tool for doing statistical analysis on a set of comparable sales in order to create a multiple regression formula that can be used to determine the most probable sales price of a subject property (or properties).

To begin, click the green Create a new model button. Enter a name and select the state of your comparable sales and subject property. Models that you have previously created will appear on this page.
