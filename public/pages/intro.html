The Mass Appraisal GIS provides an easy tool for doing statistical analysis on a set of comparable sales. A multiple regression formula is created to determine the most probable sales price of a subject property (or properties).

Comparable data can be tabular or spatial. Check with county governments to get parcel data in ESRI Shapefile format. Spatial data has the advantage that additional fields can be added from the USDA soils database.
